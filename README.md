# Major-Project
SEES: A Scalable and Energy Efficient Scheme for green IoT-based heterogeneous nodes.
Link to SEES official paper: https://drive.google.com/open?id=0B6DocT_qEp8cUEw1X2xZeXR0aTRDbVBNelJEODhOWlhjc3pJ
